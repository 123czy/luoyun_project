# Bot Chain AI Agent Launchpad — 项目记忆

## 这是什么
Bot Chain 上「会自己赚钱、自己付推理」的 AI agent 的发射台 + 自养结算层(BSC fork 链)。
在 meme launchpad 之上加一层面向 agent 的机器可读接入层 + 收益归集层 + 钱包与支付层。

## 当前任务(本仓库 luoyun_project)
评估 luoyun_project 能否作为「第一个旗舰 agent:AI 虚拟人」的底座
(自主运营社交媒体、吸粉、形成类 meme 社区凝聚力)。
评估维度见 `docs/PROJECT_CONTEXT_HANDOFF.md` 第 7 节,给出「接着做 / 部分复用 / 重起」的结论。

## 背景文件(请先读)
- `docs/PROJECT_CONTEXT_HANDOFF.md` —— 旗舰决策 + 开源选型 + 风险(从 claude.ai 讨论带过来)
- `docs/` 下三份原始调研:项目理解、基建清单、赛道调研与开发方案(需我手动放进去)

## 几条不要忘的原则
- 核心卖点是「自养」:旗舰必须有真实、链上可审计的收入(x402 服务费/赞助),不能只靠代币交易费,
  否则退回成"套 AI 皮的 memecoin"。
- 「完全自主」早期不要无人值守:内容要有护栏 + 人工兜底。
- 不重造轮子:底层发币复用 four.meme 仿盘,人力压在 Layer B(钱包+分账+推理轨道+MCP)。
- 开源栈参考:大脑/社交用 ElizaOS;视觉/直播可选 Open-LLM-VTuber 等;结算用平台自有 Layer B。
